Blocbox web application

Project optimized for Chrome & Safari

Goal: Design an application which allows people to collect information and store it in a single place,
to create simple documents, store images, links to websites, and collaborate with others.

Process: Create personas, user flows, low- and high-fidelity wireframes, design a brand identity,  mockups, create an interactive prototype, usability testing, develop Home and Dashboard pages.

Skills: Draw.io, Balsamiq, Illustrator, Sketch, Photoshop, InVision, UsabilityHub, HTML/CSS.
